Sistema de Emails
¿Cómo funciona el envío de emails?
Cuando un chofer genera un albarán, el sistema envía automáticamente dos emails:
1. A la oficina de Barroso (excabarroso@gmail.com) con todos los datos del albarán.
2. Al cliente destinatario con el PDF del albarán adjunto.

Ambos emails incluyen: número de albarán, fecha y hora, chofer y matrícula, cliente, materiales entregados con cantidades, obra/destino, ubicación GPS y firma digital.
⚠ Para activar los emails necesitas configurar EmailJS (servicio gratuito hasta 200 emails/mes).
Regístrate en emailjs.com con el email excabarroso@gmail.com y obtén los tres códigos siguientes:
Public Key (Account → API Keys)
Service ID (Email Services)
Template ID (Email Templates)
